Regression
##########
 
A regression problem is a supervised learning problem where the output variable is a real or continuous value, such as “salary” or “weight”. 
Many different models can be used to solve a regression problem, the simplest is the linear regression which tries to fit data with the best hyper-plane which goes through the points.
