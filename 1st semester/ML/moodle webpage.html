  • Description

    Description

    The aim of this course is to give the fundamentals of machine learning, with a particular emphasis on how learning algorithms are constructed.

    Each course is divided in two parts of 3h : a course part, where we describe a method, and another 3h part where either some details about the construction of the method, or a practical session using this method is provided.

    All practical sessions are done using R.

    Always come with your laptop, with Rstudio installed on it

    https://www.r-bloggers.com/in-depth-introduction-to-machine-learning-in-15-hours-of-expert-videos/

    2018 / 10 / 24 Course 1

    In this course, we will start with the ubiquitous dimensionality reduction method PCA. We will focus on interpretation rules with practical cases. 

    The correction of the lab is available in the folder lab. The supplementary exercices are available. Main reference for PCA is the chapter PCA of the book exploratory analysis with R.

    2018/ 11 / 6 Course 2

    In this course, we will present two main clustering algorithms: K-means and AHC.

    The correction of the lab is available in the folder lab. The unrated supplementary exercises about K-means derivation and implementation and AHC by hand is also available. References for this lecture are  the chapter clustering of the book Exploratory Multivariate Analysis by Example using R, and the chapter clustering of Introduction to Statistical Learning with Applications in R.
