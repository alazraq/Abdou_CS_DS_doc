# Password management

This article describes best practices in terms of password management.

## KeepassXC

- KeepassXC is a password manager which is free, cross-platform and open source. It stores all your passwords in an encrypted database.
- The database is a ``.kbdx`` file which can be versionned with git.
- You need a single password to access the .kbdx database, and then you can add or check passwords.
